Usage: agent-toolkit-cli [OPTIONS] COMMAND [ARGS]...

  agent-toolkit-cli — metadata-aware commands for asset frontmatter and
  AGENTS.md.

  Resolves the toolkit repo via: --toolkit-repo flag > AGENT_TOOLKIT_REPO env
  > .agent-toolkit-source walk-up > ~/GitHub/agent-toolkit default.
  Subcommands' --toolkit-repo flag still works for explicit per-invocation
  paths.

Options:
  --toolkit-repo DIRECTORY  Path to the agent-toolkit repo (overrides
                            env/walk-up/default).
  --project DIRECTORY       Path to the consumer project (for
                            link/unlink/list/diff).
  --help                    Show this message and exit.
