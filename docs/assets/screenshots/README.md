# Screenshots

Drop real TUI screenshots here, then replace the placeholder admonition in
[`docs/agent-toolkit/tui.md`](../../agent-toolkit/tui.md) with the matching
image embed.

Naming: `tui-<view>.png` — e.g. `tui-skill-grid.png`, `tui-scope-toggle.png`,
`tui-filter.png`, `tui-pending-apply.png`.

To embed one (the placeholder blocks name the intended file):

```markdown
![Skill grid](../assets/screenshots/tui-skill-grid.png)
```

This directory is published with the site (it is **not** under an
`exclude_docs` path), so anything committed here ships.
